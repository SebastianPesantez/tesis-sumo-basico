# Tesis – Simulación y control de tráfico vehicular con SUMO

Este repositorio contiene el primer trabajo desarrollado para el **trabajo de titulación**, enfocado en la simulación y control del tráfico vehicular utilizando el simulador **SUMO (Simulation of Urban MObility)**.

## Objetivo general
Desarrollar y evaluar estrategias de simulación y control de tráfico vehicular mediante SUMO, orientadas a escenarios urbanos reales.

## Estructura del repositorio

- `docs/`  
  Documentación técnica, notas y bitácora del desarrollo.

- `scenarios/`  
  Escenarios de simulación (redes, rutas, configuraciones).

- `scripts/`  
  Scripts en Python (TraCI, análisis, control).

- `results/`  
  Resultados de simulaciones (resúmenes y ejemplos, no datos pesados).

## Entorno de trabajo

- SUMO compilado desde código fuente
- Ubuntu 24.04 sobre WSL2
- Python 3.12 con entorno virtual

> Nota: SUMO y sus dependencias no se incluyen en este repositorio.
